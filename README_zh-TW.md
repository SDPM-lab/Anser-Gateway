# Anser-Gateway

## 初次使用

利用docker-compose指令建構 Anser-Gateway 容器依賴套件
```docker-compose
docker-compose build
```

## 啟動容器並進入

```docker-compose
docker-compose up -d
```
```docker-compose
docker-compose exec app bash
```
```command
composer install
```

## 可用指令

於專案根目錄中使用以下指令啟動伺服器:

```command
php anser serve start
```
